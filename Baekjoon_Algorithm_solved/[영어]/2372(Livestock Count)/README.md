# 문제

Print a table that describes the current count of all your livestock.

# 출력

Print the table below as shown. The character “-”, is a dash not an underscore.

```
Animal      Count
-----------------
Chickens      100
Clydesdales     5
Cows           40
Goats          22
Steers          2
```
